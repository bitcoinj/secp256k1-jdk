# Security Policy

## Supported Versions

As `secp256k1-jdk` is only available as a _pre-release_, there are no officially supported version(s) of `secp256k1-jdk` at this time.
However, we do take security seriously and will accept and respond to vulnerability reports.

| Version | Supported          |
| ------- | ------------------ |
| < 1.0   | :x: (see above)    |

## Reporting a Vulnerability

To report a vulnerability please use "Report a vulnerability" on the secp256k1-jdk [Security](https://github.com/bitcoinj/secp256k1-jdk/security) page.
